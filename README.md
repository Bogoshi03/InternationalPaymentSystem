# Secure International Payment System

## Description
This project is a secure customer portal for an international banking system. It implements authentication, input validation, and protection against common web security threats.

## Features
- Password hashing using bcrypt
- Input validation and sanitisation
- JWT authentication
- Protected routes
- Rate limiting and attack protection
- HTTPS awareness implementation

## Technologies Used
- React (Frontend)
- Node.js & Express (Backend)
- JWT Authentication
- Helmet Security
- Express Rate Limit

## How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start  

## Security Features
- Password hashing
- Protection against brute-force attacks
- Input validation
- Secure headers using Helmet
